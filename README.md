## Tech Stack

- **Framework**: Next.js 15 with React 19
- **Styling**: TailwindCSS
- **Animation**: Framer Motion
- **Database**: PocketBase
- **Deployment**: Netlify

## Quick Start

```bash
# Install dependencies
pnpm install

# Run development server
pnpm dev

# Build for production
pnpm build
```
